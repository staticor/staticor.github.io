矩阵基本

矩阵(matrix) 是一组数的排列, 如果是一维排列就是一个向量:

按成行成列的形式, 在它之前我们可能最早接触的这种排列形式的概念是方程组.
而最简单的方程就是线性方程, …

你在很多的线性代数中都会发现矩阵就是从线性方程组开始引起的.

基本概念

  • 矩阵行变换

    对矩阵中的一行 乘以一个非零倍数k;
    调换任意的两行;
    对矩阵任意两行 进行相加或相减, 替换原来中一行;

  • 线性无关

如果 向量组 $ \alpha_1 \alpha_2 \alpha_3 … … \alpha_r $ 形成的线性方程

$ k_1 \alpha_1 + k_2 \alpha_2 + … + k_r \alpha_r = 0 $

在给定数域内只有零解, 则这r个向量组成的向量组称作线性无关

  • 极大线性无关

由s个向量组成的向量组最多只能找出r个向量, 它们构成了线性无关向量组, 这r个向量称作原来向量组的极大线性无关组, 它并不唯一.

通过矩阵行变换的方式可以很快得到极大线性无关组 (非零首元, 也记作 主元 pivots)

  • 秩: 矩阵A行(列)向量线性无关

  • 满秩分解 $s n$矩阵A的秩为r 证明存在 $sr$矩阵B和$r*n$矩阵C, 使得 $A=BC$

在线性代数里, 有几种分解表达式非常重要.

分别是:

$ A = LU $ : 表示A经过行变换得到阶梯型 (erechlon form)

$ A = QR $ : Q是正交矩阵; A是可逆矩阵;

额外备注: 将一组线性无关的向量组构造成一组标准正交基的方法叫做 Gram-Schmidt 正交化.

A 的对角化 $ A = S \Gamma S^{-1} $

当A是实对称矩阵时, 进一步地能找到 正交矩阵

这里的一系列概念离不开对特征值 和特征向量 (eigenvalues, eigenvectors)的熟练掌握.
$ A = Q \Gamma Q^T $

A 的 SVD分解 $A= U \Sigma V^T $
这里对A的要求降低了, 它是一个长方形矩阵(也可以) !

行空间+零空间中找到 n个线性无关的基向量 $ v_1, v_2, … v_n $
构造的正交矩阵 V ;
在列空间+左零空间中找到m个线性无关的基向量 $ u_1, u_2, … u_m $
构造的正交矩阵 U;

求解U和V的方式是借助前面的铺垫, 求V借助 $ A^T A $ 的对角化.
求U借助 $ A A^T $

线性空间

向量空间V, 数域F 若满足 V内加法的封闭性和V对F数乘的封闭性, 则定义V在F上的线性空间.

基本性质: 零元唯一, 负元唯一 …

必须要掌握的概念有, 基、维数

基是能够线性表示线性空间任意向量的一组向量组， 且是线性无关向量组。
称这组向量是线性空间V的一组基。 基的个数称作这个空间的维数。

特殊的线性空间：

  • 零空间 线性方程 Ax = 0 的解空间

  • 无限维空间 代表， 定义在实数域上 关于x的多项式全体 F[x]

  • 矩阵列空间 矩阵行空间

  • 转置矩阵的对应列空间，行空间，零空间

描述某线性空间就要知道他的基本信息: 维数, 基向量组 是什么样的.

行空间和零空间是什么关系>

正交

正交的概念太过重要. 我深刻感到过去十年对线性代数的认知处于大学水平, 最大的困惑就是来自没有跳出对正交的初级漩涡…

正交, 即是垂直的数值定义.
空间中向量的正交首先要明确是在实数域还是复数域

如果是实数空间, 向量正交就是 inner product (dot product) 的值为0.

实对称矩阵拥有不同一般的性质 它可角化, 而且它的所有特征值都是实数.

一般情况, 对于任意方阵, 要解一个n次方程, 是可能会出现复数根的!

但实对称矩阵却保证了这一点不会出现, 因为利用实数对称的性质, 能够证明

即 特征值的虚部为 零 !

那么在复数空间呢? 就要了解共轭转置了.

$ A = A^H = A的共轭转置 $ 这种矩阵记作 Hermitian 矩阵 (埃尔米特矩阵, 对应于实对称矩阵)

矩阵应用

数量表征

首先将现实中的信息进行数量化, 结构存储到计算机, 在很多场景下就是借用一个多维数组实现的 (multi-array)

以图片为例, 我们或许知道图片是由若干个点组成的. 也就是平常说的4000万像素, 那么Pixel进一步是什么呢?

颜色RGB 三元组就是个三维向量,分别由0-255 三个整数来描述在颜色空间中的深浅程度.

所以如果是100100的图片, 就是100100*3 这样的一个多维度颜色矩阵了.

在图论中, 最重要的一种表达方式就是定义图中的各个结点和各条边.

如果有m条边, n个结点, 那么就得到一个m*n矩阵,
每行的边分别用 -1 和 1 (第i列和第j列)对应这条边的出度和入度.
