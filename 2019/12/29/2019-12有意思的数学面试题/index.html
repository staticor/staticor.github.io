12生肖/12星座优惠券问题

有个餐馆，每天随机发印有12生肖中的一种生肖的优惠券，小明每天都去该餐馆。问小明期望需要几天能收集到所有12生肖图案的优惠券。

解： 变量声明如下
$X_i$表示收集到第i种所需天数.
S表示收集12种所用天数， 即 $S = X_1 + X_2 + … X_{12}$

  • $X_1$ = 1
  • 考虑$X_2$， 除了第一种以外的优惠券，针对小明来说都是新的第二种，所以他获得第二种的优惠券的概率是 $\frac {11} {12}$, 期望天数是 $\frac {12} {11}$
  • …
  • 考虑$X_{12}$, 除了前十一种以外的优惠券，针对小明来说都是新的第十二种，所以他获得第二种的优惠券的概率是 $\frac {1} {12}$, 期望天数是 $\frac {12} {1}$
  • 利用性质： 随机变量的和的期望等于期望之和
    E(S)=E(X1)+E(X2)+…+E(X12) = 12(1+1/2+1/3+..1/12)
    可直接求和， 约为37.2天, 向上取整数, 期望38天。
