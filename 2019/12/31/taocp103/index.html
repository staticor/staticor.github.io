习题16[M22]

不用数学归纳法, 证明

证明: 错位相减的形式推导.

令S = 左 = $ \sum_{j=1}^n j x^j $
$x \ne 1$, $ T = S x = \sum_{j=1}^n j x^{j+1} $
使用变量替换, $ T = n x^{n+1} + \sum_{k=1}^n (k-1)x^k $
得,
$S - T = (1-x)S = \sum_{j=1}^n x^j - nx^{n+1} + x $
化简后为结果.

习题23[10]

当没有整数满足R(j)时， $\sum_{R(j) } a_j 和 \prod_{R(j)} a_j $分别定义为0和1 。

这是显然的。 如果不这样做， 和式和积式再参与其它运算会改变结果。
而若参与运算， 0不会改变和式结果， 1 不会改变积式结果， 故有此定义。
