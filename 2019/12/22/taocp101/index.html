题目解答

习题1 [10]

给出了置换操作[t <- m, m <- n, n <- t], 那么对于给定输入四个变量(a,b,c,d)重新排列成(b,c,d,a)， 要最少进行多少赋值操作？

解： 需要进行五次赋值操作。 ==>
t <- a, a <- b, b <- c, c <- d, d <- a
可推广至 n-tuple 情况， 最少要进行 n + 1 次。

习题2 [15]

证明从步骤E1 第二次执行执行起，每次该步开始时，m 必须大于n (第一次执行时可能不满足);

证明: 由于一开始无法确认 的大小关系， 因此从第二次开始执行时会确保 m > n
『余数的定义』：
令a， b均为整数， b不为0 定义： divmod(a, b) = (p, q) 则有 bp + q = a
且 余数q是 [0, b) 的整数
=证毕=

习题3 [20]

（为了提高效率）修改算法E，使其避免出现 m <- n 之类的 平凡替换操作， 按照算法E的风格写出这个新算法， 称为为算法F。

根据要求， 尽量避免m,n变量的交换， 可以直接进行计算结果的赋值。书写:

  • F1:[求 m/n 余数] 用 n 除 m, 令 m 为余数；
  • F2:[检验 余数是否为0?] 如果m = 0, 算法终止， n是所求结果.
  • F3:[求 n/m 余数] 用 m 除 n, 令 n 为余数
  • F4:[检验：余数是否为0？] 如果 n = 0, 算法终止， m是所求结果. 否则返回F1.

习题4 [16]

2166 与 6099的最大公因数是多少？

解: 57， 57(38, 107) = (2166, 6099) 。 最大公因数是57

习题6 [20]

当n = 5时， 执行算法步骤E1的平均次数 $T_5$ 是多少？

解: n = 5 时， m 的取值可能只有5种。(k是整数)

  • 5k
  • 5k + 1
  • 5k + 2
  • 5k + 3
  • 5k + 4
    对应这5类情况， 出现概率均为 $\frac {1} {5}$. 而它们进行E算法， 要进行E1的次数分别是 1（ 一步到位 余数为0）、2 （多进行一次 5除以1）、3（多进行2次 5除以2， 2除以1）、4（多进行2次 5除以3， 3除以2， 2除以1）、3（多进行2次 5除以4，4除以1）。

因此平均次数是 $ \frac {1+2+3+4+3} {5} $ = 2.6 次。
=解毕=

习题7 [M21]

m已知， n在所有正整数范围内取值，令Um为算法E中步骤E1执行的平均次数。 说明Um具有合理定义， Um与Tm有关系吗？

证明: $U_m $ = $T_m + 1$

首先回顾书中给出Tm的定义:
假设n值已知， m取值在所有正整数上， 执行E算法平均E1要执行的次数
（除了第一次情况之外，后面的场景其实只要考虑n的余数情况。 可以代入一点群论的思想）

这个题目我的突破启发于数学极限的 $N- \epsilon 定义$。 即研究的是一个极限问题，那么我就不考虑m是多少， 只针对n在所有正整数范围取值这句话， 那么一般性问题就是n很大（远大于m的情况）。

当n大于m时，E算法其实经过第一次E1后，后面问题我发现就转变成了 “n已知， m发生变化”的问题， 因此这个题目研究的 $U_m$ 也就得出了。

=证毕=

通过指定式（3） 中 $\theta_j, \phi_j, a_j, b_j$ 给出计算正整数m和n最大公因数的“可行的”形式。
算法： 令输入由字符串 $a^mb^n$表示， 也就是在m个a后连着n个b。 你的解法应当力求尽可能简单。

提示： 利用算法E， 把步骤E1的除法改为 置 $ r \leftarrow |m-n|, n \leftarrow min(m,n) $

习题9 [M30]

假定 $C_1 = (Q_1, I_1, \Omega_1, f_1) $ 和 $C_2 = (Q_2, I_2, \Omega_2, f_2) $ 是两个计算方法. 例如 $C_1$ 可表示这一节的算法E, 不过要限制 m 和 n大小; 而$C_2$则表示算法E的一个计算机程序实现. (因此 $Q_2$可以是计算机所有状态的集合), 也就是计算机内存和寄存器所有可能的配置: $f_2$可以是计算机单步操作的定义; $I_2$ 可以是所有初始状态集合, 包括确定最大公因数的相应程序以及m 和n的值)

试就 “ $C_2$ 是 $C_1$ 的一个表示” 或者 “ $C_2$ 模拟 $C_1$ “的概念, 给出集合论定义.

直观上, 这意味着 $C_1$ 的任何计算序列都由 $C_2$ 模拟实现, 不过 $C_2$ 可能要用更多的计算步骤, 保留更多状态信息.

令 $P_i$表示X天， 有且仅且收集了 $i$种卡片的概率。
显然， 这里 i 取值为 1，2，3…, 12 只有12种可能。 所以：

下面分别列式：

$ P_1 = C_{12}^1 (\frac {1} {12})^X $,

$ P_2 = C_{12}^2 \sum_{\alpha_1=1 , \alpha_2=1, \alpha_1 + \alpha_2 = X} (\frac {1} {12})^{\alpha_1} (\frac {2} {12})^{\alpha_2} $

….

$ P_{11} = C_{12}^{11} \sum_{\alpha_1=1 … \alpha_{11}=1, \alpha_1 + … + \alpha_{11} = X} (\frac {1} {12})^{\alpha_1} … (\frac {11} {12})^{\alpha_{11}} $

$ P_{12} = C_{12}^{12} \sum_{\alpha_1=1 … \alpha_{12}=1, \alpha_1 + … + \alpha_{12} = X} (\frac {1} {12})^{\alpha_1} … (\frac {12} {12})^{\alpha_{12}} $
