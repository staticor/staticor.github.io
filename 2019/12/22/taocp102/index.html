表述: 首先回顾书中给出的 算法I(构造证明)

  • I1. [证明P(1)] 置 k <- 1, 并且证明命题P(n)情况为 n = 1时成立;
  • I2. [蕴含关系] 如果 P(1),P(2), …, P(n) 全部为真, 并且能证明出 P(n+1)

将上面的表述修改为 起点从0开始:

  • I1. [证明P(0)] 置 k <- 0, 并且证明命题P(n)情况为 n = 0时成立;
  • I2. [蕴含关系] 如果 P(0),P(1),P(2), …, P(n) 全部为真, 并且能证明出 P(n+1)

习题6[10]

令x = $m+0.d_1d_2 … $ , y = $ n+0.e_1e_2 … $ 为实数. 基于十进制表示,给出用来判定 x, y 大小关系的规则.

解: 从高位至低位逐位判定. 这里假定m和n都表示的某整数.

  • 从第一位开始, 若 m = n 则继续, 否则 m > n 对应 x > y, m < n 对应 x < y;
  • 进行到第k位比较, 若 $d_k = e_k$, 则继续, 否则 $d_k > e_k$ 对应 x > y, $d_k < e_k$ 对应 x < y;
  • 依次类推.

习题25[22]

假设我们有一台二进制计算机和一个数x, x in [1, 2), 说明下面的算法,可计算
$ y = llog_b x $ 近似值, 仅需使用同所需精度小数位数成正比的移位\加法\减法运算:

  • L1. [初始化] y <-0, z <- x 右移1位, k <- 1
  • L2. [检验算法终结.] 如果 x = 1 终止;
  • L3. [比较.] 如果 x - z < 1, 置 z <- z 右移一位, k <- k+1 重复这一步;
  • L4. [减少值] 置 x <- x -z, z <- x 右移k位, y <- $ y + log_b (2^k / 2^k -1) $, 然后转 L2.
