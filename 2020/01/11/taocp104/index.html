习题3[M10]

令n为整数， x为实数， 证明
(a) 当且仅当 $x < n$ 时 ， $ \lfloor x \rfloor < n$;

(b) 当且仅当 $n \le x$ 时 ， $ n \le \lfloor x \rfloor $;

(c) 当且仅当 $x \le n$ 时 ， $ \lceil x \rceil \le n$;

(d) 当且仅当 $n \lt x$ 时 ， $ n \lt \lceil x \rceil $;

(e) 当且仅当 $ x - 1 \lt n \le x $时，又当且仅当 $ n\le x \lt n+1 $时， $ \lfloor x \rfloor = n $

(f) 当且仅当 $ x \le n \lt x + 1 $时，又当且仅当 $ n-1 \lt x \le n $时， $ \lceil x \rceil = n $

这几个性质都不难证明， 但手写一写会加深自己对取整函数的理解。

证明: $ \lfloor x \rfloor$ 表示的是个整数， 而且和原来数x相比，有约束。

如果两个整数m, n， $ m \lt n \leftarrow \rightarrow m\le n-1 $.
结论(a) 可以分两种情况考虑 若 $ x \ge n $, 则 $ \lfloor x \rfloor = n $ 得不出结论；
反之， $ x \lt n $, $ \lfloor x \rfloor \le x \lt n $, 得证。

(b) （c) (d) 与(a)类似。 这里略。

(e) (f) 相当对前四小问的结合形式。

习题5[16]

假定x为正实数， 建立一个把x表示成舍入到最近的整数的简单公式。

解：考虑到round函数是与0.5有关的函数， 大于0.5则取上面的整数， 小于0.5则取下面的整数， 因此将原来的值加上0.5， 再做向下取整。

$ \lfloor （x+0.5） \rfloor $

习题6[20]

在下列等式中， 哪些等式对于所有正实数x恒成立?
(a) $ \lfloor \sqrt { \lfloor x \rfloor} \rfloor = \lfloor \sqrt x \rfloor $
(b) $ \lceil \sqrt { \lceil x \rceil} \rceil = \lceil \sqrt x \rceil $
(c) $ \lceil \sqrt { \lfloor x \rfloor } \rceil = \lceil \sqrt x \rceil $

解： a b成立 平方后证明；
c不成立， 可举例证伪： x = 1.5

习题7[M15]

（取整函数的三角不等式）证明 $ \lfloor x \rfloor + \lfloor y \rfloor \le \lfloor x + y \rfloor $， 并且当且仅当 $ x \quad mod \quad 1 + y \quad mod \quad 1 \lt 1 $时取等号。 对于向上取整函数， 类似的公式成立吗？

证明： $ \because x = \lfloor x \rfloor + x mod 1 ， y = \lfloor y \rfloor + y mod 1 $
$ \therefore x + y = \lfloor x \rfloor + \lfloor y \rfloor + x \quad mod \quad 1 + y \quad mod \quad 1 $
$ \because x \quad mod \quad 1 , y \quad mod \quad 1 \in [0, 1) $
$ \therefore \lfloor x + y \rfloor \ge \lfloor x \rfloor + \lfloor y \rfloor $

当 $ x \quad mod \quad 1 + y \quad mod \quad 1 \lt 1 $ 等号易证。

对于向上取整函数， 不等式方向应改为大于等于。

习题16[M10]

假定 $ y \gt 0$. 证明: 如果 $(x-z)/y$ 是整数, 且 $ 0 \le z \lt y$, 那么 $ z = x \pmod y $.

证明: 令 $ x - z = ky, k \in \it Z $, 则根据定义

$ x \pmod y = x - y \lfloor x/y \rfloor = x - ky = z $

证毕。

根据同余定义直接证明定律A; 再证明D的一半: 如果 r s 互素， $ a \equiv b \pmod rs , 那么 a \equiv b \pmod r 且 a \equiv b \pmod s $
(r和s为任意整数)。

证明： 根据同余定义， 以及 Mod 定义， 可以得到 a - b 是 rs倍数， 又因为 r , s 互素的关系。
因此 a - b 是 r倍数， 也是 s倍数。

习题21[M22]

（算术基本定理）利用定律B，证明每个正整数 $ n \gt 1 $可以写成素数乘积的形式， 且这种表示是唯一的.

证明： 这个题目太有意思，也特别基础。
假设给定一个素数列 $ \mathit P = { p_k } $， 证明用它可以表示每个正整数 $n$， 并且表示方法唯一。 （顺序假定严格从小到大）。

可以分类讨论：
(i) 如果 $n$ 为素数， $ n \in \mathit P ， 可以假设 n = p_m $, 则直接可以得出结论;
(ii) 如果 $ n $ 不是素数， 那么必然存在一个最小的整数 $ p_1 \in \mathit P $, 使得 $ p_1 可以整除n$。
这个 $p_1$是唯一存在的，那么接下来问题变更为使用 $\mathit P $表示整除后的商；
重复以上步骤直到最终商为素数， 问题变更为第一种。

证明： 因为p是素数， 所以小于p的整数 从1到p-1 均与p互素， 所以 $\phi(p) = p-1 $

结果是 $ \phi(p^e) = p^e - p^{e-1}$
