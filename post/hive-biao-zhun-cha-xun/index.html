窗口查询

全量窗口查询

select grp, sum(money) from student group by grp

需求， 求每个组占总money的比例， 并排序。

思路1， groupBy 分别计算分子和分母

select a.*, totalmoney, a.subtotal / totalmoney from (select grp, sum(money) subtotal from student group by grp )a join (select sum(money) as totalmoney from student ) b on 1=1

注意， set hive.strict.checks.cartesian.product=flase;

//使用 全量窗口

select grp, v, sum(v) over() as total from (select grp, sum(money) as v from student group by grp )a

使用 全量窗口，性能会好于 group by.

分区窗口查询

选择一个离散categorical 字段， 对数据分区。

OVER() 内， 放置分区逻辑。

      , row_number() over(partition by player_id order by match_day asc) rk1

分区窗口内引入 BOUND

知识准备

截止当前累加的和 （阶梯累加）

select name, orderdate,  money
, sum(money) over ( 
    partition by name,   substring(orderdate, 1, 7)  order by date 
     rows between UNBOUNDED PRECEDING and  CURRENT ROW 
)  from   。。。

游标取值范围， 要传入两个参数， 默认值：
rows between UNBOUNDED PRECEDING and CURRENT ROW

当前行（ 默认值） ： CURRENT ROW
往前n行 n PRECEDING
往后n行 n FOLLOWING
起点 UNBOUNDED
起点之前（默认值） UNBOUNDED PRECEDING
终点之后 UNBOUNDED FOLLOWING

需求， 计算每个人最近7天的累积消费，截止当前付费

select name, orderdate,  money
, sum(money) over ( 
    partition by name,   substring(orderdate, 1, 7)  order by date 
     rows between UNBOUNDED PRECEDING and  2 PRECEDING
)  from   。。。
